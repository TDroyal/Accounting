## 1. 技术选型

| 类别 | 选型 | 说明 |
|------|------|------|
| 框架 | Vue 3 + TypeScript | Composition API、`<script setup>` |
| 构建 | Vite 5 | 快速 HMR、产物打包 |
| 状态 | Pinia 2 | 模块化 store |
| 路由 | Vue Router 4 | History 模式 |
| 组件库 | Element Plus（桌面） + Vant（移动）按需加载 | 或统一 Element Plus 走响应式，MVP 阶段先统一用 Element Plus 减少适配成本 |
| 图表 | ECharts 5 | 见 [07-可视化与统计设计](./07-可视化与统计设计.md) |
| HTTP | Axios | 拦截器封装 token、错误统一处理 |
| 工具 | dayjs | 日期处理，轻量 |
| 包管理 | pnpm | 速度快、节省磁盘 |

> 说明：MVP 阶段优先**统一使用 Element Plus + 响应式布局**，降低双组件库适配成本；待移动端体验不足时再引入 Vant 替换关键交互（如数字键盘、Tab 栏）。

## 3. 响应式布局

### 3.1 断点
| 断点 | 宽度 | 布局 |
|------|------|------|
| mobile | < 768px | 单列、底部 Tab、浮动记账按钮 |
| tablet | 768–1023px | 双列 |
| desktop | ≥ 1024px | 顶栏导航 + 多列、信息密度高 |

### 3.2 实现方式
- 使用 `composables/useResponsive.ts` 暴露当前断点：
```ts
export function useResponsive() {
  const width = ref(window.innerWidth)
  window.addEventListener('resize', () => { width.value = window.innerWidth })
  const isMobile = computed(() => width.value < 768)
  const isDesktop = computed(() => width.value >= 1024)
  return { width, isMobile, isDesktop }
}
```
- 顶层根据断点切换 `MobileLayout` / `DesktopLayout`；
- 关键组件用 CSS 媒体查询微调（如 `el-row` 的 `gutter`、栅格 `:xs/:sm/:lg`）。

### 3.3 路由
```
/              → 重定向到 /statistics/daily（已登录）或 /login
/login, /register
/transactions/new       快速记账
/transactions           流水列表
/statistics/daily       日统计
/statistics/monthly     月统计
/statistics/yearly      年统计
/categories             分类管理
/settings/budget        预算
/settings/accounts      账户（P2）
```
路由守卫：未登录访问受保护页跳 `/login`。

## 4. 状态管理（Pinia）

### 4.1 auth store
- `token`、`userId`、`profile`
- `login()`、`logout()`、`restore()`（从 localStorage 恢复）

### 4.2 category store
- `tree`：分类树（含缓存）
- `load()`、`add()`、`update()`、`toggleStatus()`
- `frequentlyUsed`：常用分类（按近 N 次使用频次）

### 4.3 transaction store
- `list`、`filters`、`pagination`
- `create()`、`update()`、`remove()`
- 创建后调用 statistics store 的刷新

## 5. HTTP 封装（api/request.ts）

- 请求拦截器：注入 `Authorization: Bearer <token>`；
- 响应拦截器：
  - 解包统一响应 `{code,message,data}`；
  - `code !== 0`：弹消息提示 + reject；
  - 401：清 token 跳登录；
  - 网络错误：友好提示。
- 超时与重试配置。

## 6. 关键页面设计

### 6.1 快速记账页（手机优先）
- 顶部金额大号显示 + 数字键盘（移动端可用 Vant NumberKeyboard，桌面端用普通 input）；
- 分类选择：宫格图标，常用置顶，可滑动；
- 时间默认当前，可点改；
- 提交按钮置底，防抖。

### 6.2 月度统计页（桌面优先）
- 顶部摘要卡：累计、环比、预算进度条；
- 左：每日支出折线；
- 右：分类占比柱状（降序）；
- 下方：明细列表，可按分类筛选。

### 6.3 日统计页
- 当日总额大号；
- 分类占比饼图；
- 明细列表（倒序）。

## 7. 性能与体验

- 路由懒加载 `() => import(...)`；
- ECharts 按需引入模块，减小体积；
- 列表虚拟滚动（明细量大时）；
- 移动端首屏 < 1s（LCP）。
